# Stride workout program — evidence-based Upper/Lower (advanced)

Built from Christian's equipment ([training-profile.md](training-profile.md)), grounded in
hypertrophy science + Jeff Nippard's selection logic. This is the spec the planner builds from.

## Principles (verified)
- **Volume:** advanced MEV ~12–14 sets/muscle/wk; productive **MAV ~12–20**. Cycle MEV→MAV across a block.
- **Frequency:** ~**2×/week** per muscle (Upper/Lower delivers this).
- **Reps:** hypertrophy equivalent ~5–30 reps near failure; compounds 5–8 (strength touch), isolations 12–20.
- **Effort:** **0–3 RIR** (heavy compounds 2–3, isolations 0–1); failure adds fatigue, little extra growth.
- **Rest (Schoenfeld 2016):** compounds **2–3 min**, isolations **60–90 s** (longer rest grew MORE).
- **Selection:** high stimulus-to-fatigue, stable, **stretch-biased** (overhead triceps, incline, deep-stretch legs).
- **Deload:** every **4–8 wks**, ~50–70% volume/intensity for a week.

## Weekly structure
**4-day** (each muscle 2×/wk): Mon Upper A · Tue Lower A · Thu Upper B · Fri Lower B · cardio ×2 on off days.
**5-day** (upper-skewed, upper recovers faster): Mon Upper A · Tue Lower A · Wed Upper B · Fri Lower B · Sat Upper C (weak-points: side delts, arms, pull-ups, chest). Rotate start each week.

## Sessions (ordered · sets×reps · RIR · rest)
**Upper A (strength-leaning push):** Bench 4×5–7 (2–3, 2.5–3m) · Weighted pull-up 3×6–9 (2, 2–3m) · Incline chest press machine 3×8–12 (1–2, 90s) · Seated row 3×8–12 (1–2, 90s) · Shoulder press machine 3×8–12 (1–2, 90s) · DB incline curl 3×10–15 (0–1, 60s) ⊃ DB overhead triceps ext 3×10–15 (0–1, 60s).

**Lower A (squat pattern):** Hack squat 4×6–9 (2–3, 2.5–3m) · Weighted-sack walking lunges 3×8–12/leg (2, 2m) · Leg press deep 3×10–15 (1–2, 2m) · Leg curl *or* DB SLDL 3×10–15 (1, 90s) · Calf raise 4×10–15 (0–1, 75s).

**Upper B (back/delt bias):** Weighted pull-up 4×6–9 (2, 2.5–3m) · Incline chest press machine 4×8–12 (1–2, 2m) · Chest-supported row 4×10–14 (1, 90s) · Shoulder press machine 3×10–14 (1, 90s) · DB lateral raise 4×12–20 (0–1, 60s) · Pec-deck chest press 3×10–15 (0–1, 60s) ⊃ DB incline curl 3×10–15 (0–1, 60s).

**Lower B (pendulum/hinge):** Pendulum squat 4×8–12 (2, 2.5–3m) · RDL (DB/barbell) 3×8–12 (2, 2.5m) · Leg press glute-bias 3×10–15 (1–2, 2m) · Leg curl/DB SLDL 3×12–15 (1, 90s) · Calf raise (stretch, pause) 4×12–18 (0–1, 75s).

**Upper C (5-day, weak-points):** Pull-up 3×8–12 · Incline/pec-deck press 3×10–15 · DB lateral raise 4×12–20 · Pec-deck rear flye 3×12–20 · DB incline curl 3×10–15 ⊃ DB overhead triceps 3×10–15.

## Weekly volume (4-day, direct+indirect, within MEV–MAV)
Chest ~13 · Back ~14 · Side delts ~8 · Rear delts ~6 · Biceps ~9 · Triceps ~9 · Quads ~14 · Hams ~9 · Calves ~8. 5-day pushes upper toward 16–20.

## Cardio (interference-managed)
2× **cycling/incline-walk**, 20–25 min, on rest days or **after upper** (never near lower); keep ≤30 min. Lift first if same day.

## Progression
**Double progression:** work bottom of rep range at target RIR → add reps to top across all sets → add smallest load, reset to bottom. Log every set; load is the main driver at this level. Deload at block end (4–6 wks): ~half sets, 3–4 RIR.

- **Chest:** incline chest press machine* · bench · pec-deck press · DB incline press
- **Lats (vertical):** weighted pull-up* · neutral-grip pull-up · wide pull-up
- **Back (rows):** chest-supported row machine* · DB chest-supported row · 1-arm DB row
- **Side delts:** DB lateral raise* · lean-away lateral raise · shoulder press machine
- **Rear delts:** pec-deck rear flye* · DB prone reverse flye
- **Front delts:** shoulder press machine* · seated DB press · DB incline press
- **Biceps:** DB incline curl* · DB standing curl · hammer curl · chin-grip pull-up
- **Triceps:** DB overhead extension* · 1-arm overhead ext · close-grip/pec-deck press
- **Quads:** hack squat* · pendulum squat · leg press · weighted-sack lunges

- **Glutes:** RDL* · walking lunges · high/wide leg press · deep pendulum
- **Calves:** calf raise machine* · leg-press toe press · single-leg DB raise off box
